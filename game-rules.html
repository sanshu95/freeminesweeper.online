Game Rules

  1. The game board is a grid of hidden squares.
  2. Some squares contain mines. Your goal is to reveal all the squares that do NOT contain mines.
  3. Clicking on a square reveals it. If you click on a mine, the game is over.
  4. If a revealed square is not a mine, it will display a number. This number shows how many mines are directly adjacent to that square (touching horizontally, vertically, or diagonally).
  5. If you click a square that has no adjacent mines, it will appear blank. The game will then automatically reveal all adjacent squares for you, as they are all guaranteed to be safe.
  6. You can right-click a square to place a flag on it if you suspect it contains a mine. This is for your reference only and helps prevent accidental clicks. It has no effect on winning or losing.
  7. You win the game by successfully uncovering all non-mine squares. You do not have to flag all the mines to win.
